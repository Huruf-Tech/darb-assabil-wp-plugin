Integrate WooCommerce with Darb Assabil shipping service for seamless order fulfillment in Libya.

== Description ==

Darb Assabil for WooCommerce enables seamless integration between your WooCommerce store and Darb Assabil shipping service in Libya. This plugin automates shipping calculations, order processing, and tracking management.

== Features ==

1. Automated shipping rate calculations for Libya
2. Custom shipping zones and methods
3. Real-time order synchronization with Darb Assabil
4. Webhook integration for order status updates
5. City and area selection dropdowns for Libya
6. Order tracking integration
7. Customizable shipping settings
8. Admin dashboard for managing orders and settings

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/darb-assabil` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure WooCommerce is installed and activated
4. Navigate to WooCommerce > Settings > Shipping to configure the shipping zones
5. Go to Darb Assabil settings in the WordPress admin menu to complete the setup

== Requirements ==

* WordPress 5.5.0 or higher
* WooCommerce 3.0 or higher
* PHP 7.1 or higher

== Configuration ==

1. Add your Darb Assabil credentials in the plugin settings
2. Configure shipping zones in WooCommerce and add Darb Assabil as shipping method
3. Set up webhooks for order status synchronization
4. Customize shipping options and rates as needed

== Frequently Asked Questions ==

= Does this plugin work outside Libya? =

No, this plugin is specifically designed for shipping within Libya using Darb Assabil services.

= Is WooCommerce required? =

Yes, WooCommerce must be installed and activated for this plugin to work.

= Can I customize shipping rates? =

Yes, you can configure custom shipping rates through the WooCommerce shipping zone settings.
